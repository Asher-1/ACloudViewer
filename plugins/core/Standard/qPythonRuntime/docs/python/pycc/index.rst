pycc (Python)
=============


pycc is the module that wraps ACloudViewer's data structures to make them
available in Python.

It also define classes and functions that allows to interact with the "ACloudViewer runtime"
(that is, the currently running ACloudViewer.exe which will use the python interpreter to run rou code).
