CloudViewer-ML
==============

.. image:: https://raw.githubusercontent.com/Asher-1/CloudViewer-ML/main/docs/images/getting_started_ml_visualizer.gif
    :width: 640px
    :align: center

CloudViewer-ML is an extension of ACloudViewer for 3D machine learning tasks. It builds on
top of the ACloudViewer core library and extends it with machine learning tools for
3D data processing.

Check it out at `CloudViewer-ML <https://github.com/Asher-1/CloudViewer-ML>`_.
