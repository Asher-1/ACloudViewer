Benchmark Examples
==================

Python examples for benchmarking various algorithms and operations.

Overview
--------

The benchmark module provides tools for performance evaluation:

- **Feature Matching**: Benchmark feature matching algorithms
- **Nearest Neighbor Search**: Benchmark nearest neighbor search algorithms
- **Registration**: Benchmark registration algorithms (RANSAC, FGR)
- **TSDF Integration**: Benchmark TSDF volume integration
