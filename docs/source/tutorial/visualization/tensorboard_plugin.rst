TensorBoard Plugin
===================

.. only:: html

   .. mdinclude:: ../../../tensorboard.md

.. only:: not html

   CloudViewer provides a TensorBoard plugin for visualizing 3D geometries in TensorBoard.

   The plugin allows you to log 3D point clouds, meshes, and other geometries to TensorBoard for interactive visualization alongside your training metrics.

   For more information, see the :doc:`../../python_api/cloudViewer.visualization.tensorboard_plugin.summary` API documentation.
