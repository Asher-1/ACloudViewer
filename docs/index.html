关于 ACloudViewer

ACloudViewer 是一个开源的3D点云和三角网格处理软件库。它支持快速开发处理3D数据的软件， 高度基于 CloudCompare、Open3D、ParaView 和 COLMAP，并集成了PCL库。

最初设计用于比较两个3D点云（如激光扫描获得的点云）或点云与三角网格之间的差异。 它依赖于针对此特定用例高度优化的八叉树结构，能够处理海量点云数据 （通常超过1000万点，在2GB内存下最多可达1.2亿点）。

3D数据结构

强大的3D数据结构和处理算法，支持点云、网格和各种几何体

场景重建

基于COLMAP的场景重建系统，支持从图像到3D模型的完整流程

表面对齐

高精度的点云配准算法，包括ICP、RANSAC等多种方法

3D可视化

基于VTK和OpenGL的高性能渲染引擎，支持PBR物理渲染

机器学习

集成PyTorch和TensorFlow，支持3D深度学习应用

GPU加速

核心3D操作的GPU加速，支持CUDA 12.x

C++ & Python

提供C++和Python双语言API，灵活易用

插件系统

丰富的插件生态系统，支持自定义功能扩展

快速开始

Python 快速入门

1

安装

pip install cloudviewer
2

验证安装

python -c "import cloudViewer as cv3d; print(cv3d.__version__)"
3

运行示例

import cloudViewer as cv3d # 创建球体网格 mesh = cv3d.geometry.ccMesh.create_sphere() mesh.compute_vertex_normals() # 可视化 cv3d.visualization.draw(mesh, raw_mode=True)

C++ 快速入门

1

克隆仓库

git clone --recursive https://github.com/Asher-1/ACloudViewer.git
2

构建项目

cd ACloudViewer mkdir build && cd build cmake .. make -j$(nproc)
3

运行示例

./bin/ACloudViewer

详细编译说明请参考 BUILD.md
