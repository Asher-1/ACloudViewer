# ACloudViewer CLI Harness

The CLI harness is maintained in **CLI-Anything**, same layout as `blender/agent-harness/`:

| Item | Location |
|------|----------|
| Package source | `CLI-Anything/acloudviewer/agent-harness/` |
| Entry point | `cli-anything-acloudviewer` |
| Hub skill | `CLI-Anything/skills/cli-anything-acloudviewer/SKILL.md` |
| Install | `pip install git+https://github.com/Asher-1/CLI-Anything.git#subdirectory=acloudviewer/agent-harness` |

See the [unified documentation](../README.md) for CLI reference, MCP setup, and testing.
