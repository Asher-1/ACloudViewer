Nearest 

No filtering. Nearest neighbor is used.

Mip-mapping is activated. But no filtering occurs.

LinearMipmapNearest

NearestMipmapLinear

LinearMipmapLinear 

Both interpolated Mip-mapping and linear filtering are used.
