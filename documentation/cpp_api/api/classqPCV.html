Detailed Description

Wrapper to the ShadeVis algorithm for computing Ambient Occlusion on meshes and point clouds "Visibility based methods and assessment for detail-recovery", M. Tarini, P. Cignoni, R. Scopigno Proc. of Visualization 2003, October 19-24, Seattle, USA. http://vcg.sourceforge.net/index.php/ShadeVis
