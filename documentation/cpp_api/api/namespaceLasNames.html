Detailed Description

This namespace regroups constants for all the names we use in ACloudViewer's ScalarField system for the standard dimensions defined by the LAS Spec.

Notice that RGB and Waveforms are missing, that is normal as they are not treated as scalar fields within ACloudViewer.
