Creates a new sub mesh with the selected vertices only.

This method is called after a graphical segmentation or by ccMesh::createNewMeshFromSelection. It creates a new sub-mesh structure with the vertices that are tagged as "visible" (see ccGenericPointCloud::visibilityArray). This method will also update this sub-mesh if removeSelectedFaces is true. In this case, all "selected" triangles will be removed from this sub-mesh's instance. 

    \param removeSelectedFaces specifies if the faces composed only of
